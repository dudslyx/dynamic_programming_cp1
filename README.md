# 🚚 Logística Inteligente: Otimização de Fluxos e Fretes:
### 📋 Sobre o Projeto: 
Este projeto foi desenvolvido como parte da disciplina de Dynamic Programming. O objetivo é simular um cenário real de logística no Brasil, aplicando conceitos de eficiência algorítmica e estruturas de dados dinâmicas para resolver problemas de priorização e fluxo de carga. Cenário Escolhido: [Inserir aqui: "Simulação de Centro de Distribuição (Final PAR)" OU "Sistema de Fretes e Entregas (Final ÍMPAR)"]
### 🛠️ Tecnologias e Estruturas: 
Utilizadas Para garantir a performance e organização, o sistema utiliza:Listas (lista): Armazenamento dos dados brutos de pedidos/fretes.Tuplas (tupla): Garantia de imutabilidade para dados sensíveis (IDs, coordenadas). Dicionários (dict): Busca rápida de informações de clientes ou produtos.Deques (collections.deque): Gerenciamento da fila de prioridade com complexidade $O(1)$ para inserção e remoção. Pandas (DataFrame): Estruturação dos dados para geração de relatórios e análises. Recursão: Utilizada no algoritmo de [Ex: busca de pacotes ou cálculo de frete composto].
### 📈 Análise de Eficiência: 
(Big O) O projeto foca em manter a escalabilidade do sistema:Ordenação: Implementada com $O(n \log n)$ para garantir rapidez mesmo com grandes volumes de dados. Busca/Acesso: Utilização de tabelas hash (dicionários) para acesso em tempo constante $O(1)$.
### 📊 Visualizações: 
O sistema gera gráficos automáticos utilizando Matplotlib e Seaborn para demonstrar:Status da fila de espera.Custo total de frete por região. Nível de criticidade das entregas.
